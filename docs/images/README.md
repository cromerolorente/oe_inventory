# Imágenes del Manual de Usuario

Aquí van las capturas de pantalla que referencia [`../MANUAL_USUARIO.md`](../MANUAL_USUARIO.md).

Guarda cada captura con **exactamente** el nombre indicado en la tabla del manual
(apartado **Anexo A: Guía de capturas**), por ejemplo `05-devices.png`. En cuanto
estén aquí, las imágenes aparecerán automáticamente al abrir el manual en GitHub
o en cualquier visor de Markdown.

Recomendaciones:
- Formato **PNG**, ancho aproximado **1400 px**.
- Usa **datos de ejemplo**, no información real de personas.
- Mantén los nombres en minúsculas y con guiones, tal cual la guía.
